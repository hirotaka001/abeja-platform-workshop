Deploy したモデルを試してみよう
Http Service にある Endpoint が実際にアクセスできる Http Service の識別子になり、その横にある"目"のボタンをクリックすると、 Url と一緒に curl コマンドでのアクセス方法・アクセス例が表示されます。

4-1

あるいは、コンソールからでもチェックツールを利用してテストすることができます。
右側にある [ Check ] ボタンをクリックしてください。

4-2

以下のような Test HTTP Service 画面が表示されます。

4-3

既存の用意された画像で試して頂くことも可能ですし、タブを切り替えて、ローカル PC 内のファイルを利用したり (Upload File)、 Json テキストの直接入力を行う (JSON) 等も可能です。
今回は画像の分類を行う Model なので、 Image タブの画像や、Upload File タブからローカル PC 内の別の画像をアップロードして試してみてください。
